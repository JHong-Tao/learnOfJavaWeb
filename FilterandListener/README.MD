# FilterANDlistener
## JavaWeb三大组件
1. Servlet
2. Filter:过滤器  
3. Listener:监听器
## Filter:过滤器
    1.概念：
        - web中的过滤器：当访问服务器资源的时候，过滤器可以将请求拦截下来，完成一些特殊的功能
        - 过滤器的作用：
            1.一般用于完成通知操作，如：登录验证，统一编码处理，敏感字符处理
    2.快速入门：
        - 定义过滤器的步骤：     
            1.定义一个类，实现Filter接口
            2.重写方法
            3.配置拦截路径
                1.web.xml配置方式
                2.注解的配置方式：
                    @WebFilter("/*") :括号内的路径为需要过滤的路径
